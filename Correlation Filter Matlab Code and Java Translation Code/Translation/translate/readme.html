Android Dependencies

The following are the jars from the dependencies folder required for android applications or a newer compatible version:
  • google-http-client-android2-1.7.0-beta.jar (only for SDK >= 2.1)
  • google-http-client-1.7.0-beta.jar
  • jsr305-1.3.9.jar
  • gson-2.1.jar
  • jackson-core-asl-1.9.4.jar
  • protobuf-java-2.2.0.jar
  • commons-codec-1.6.jar
  • google-api-client-android2-1.7.0-beta.jar (only for SDK >= 2.1)
  • guava-11.0.1.jar
  • google-api-services-translate-v2-rev2-1.4.0-beta.jar
  • google-api-client-1.7.0-beta.jar
  • google-oauth-client-1.7.0-beta.jar
  • google-http-client-android3-1.7.0-beta.jar (only for SDK >= 3.0)
Please see the Android wiki for the Android Developer's Guide.
